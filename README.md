# HO-GCN
HO-GCN: Human-Object Adaptive Topological Interaction GCN for Escalator Dangerous Behavior Recognition
Recognizing dangerous behaviors on escalators is of great importance for public safety, yet remains challenging in real surveillance scenarios due to the complex interactions between passengers and escalator structures. Existing skeleton-based Graph convolutional networks~(GCNs) that mainly rely on human skeletal keypoints are insufficient for modeling human-escalator interaction topology, which limits their recognition performance on escalator dangerous behaviors. To address this issue, we propose a Human-Object Adaptive Topological Interaction GCN~(HO-GCN) for escalator passenger dangerous behavior recognition. Specifically, we represent human skeletal joints together with escalator keypoints in a unified human-object topology graph~(HO-Graph), which explicitly encodes both human body structure and escalator spatial structure. Based on the constructed HO-Graph, we further develop an adaptive interaction topology learning module to learn human-escalator interaction topology. This module refines a learnable shared topology via two complementary relation modeling branches, namely a difference-based branch and a correlation-based branch. In this way, the proposed method can effectively capture diverse topological interaction patterns between human joints and escalator keypoints. Experiments on the self-collected ESC-Danger dataset show that the proposed HO-GCN achieves an accuracy of 93.7\%, demonstrating its effectiveness for dangerous behavior recognition on escalators.
